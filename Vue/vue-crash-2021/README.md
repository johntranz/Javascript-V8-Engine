# vue-crash-2021

## Project setup
```
npm install
```

### Compile and initialize database json-server backend 
```
npm run backend
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Config the serve to global
```
npm i -g serve 
```

### Run app for production
```
serve -s dist
```
